# Datasets

Download round data from the IMC Prosperity platform and place in the corresponding round directory.

```
datasets/
  round1/   <- CSV files for round 1
  round2/
  ...
```

Dataset files are gitignored (large binary data). Each team member downloads their own copy.

## Where to Get Data

1. **Prosperity Platform** — Download from your submission page after each round
2. **Community Shared** — Check the Prosperity Discord #resources channel
3. **Backtester Repo** — Some sample data may be included in the backtester submodule
